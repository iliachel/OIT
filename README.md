## Описание проекта
Многостраничный сайт, размещённый на GitHub Pages, посвящённый анализу
концепции TinyML и сравнению популярных фреймворков для микроконтроллеров.

## Используемые технологии
- HTML5
- CSS3
- JavaScript
- AOS (Animate On Scroll)
- GitHub Pages

## Локальная разработка

Поскольку проект использует `fetch` API для динамической загрузки общего заголовка (`header`), его нельзя просматривать, просто открыв `index.html` в браузере из-за политики безопасности CORS.

Для корректной работы всех скриптов необходимо запустить локальный веб-сервер.

1.  Убедитесь, что у вас установлен Python 3.
2.  Откройте терминал в корневой папке проекта.
3.  Выполните команду:
    ```bash
    python3 -m http.server 8000
    ```
4.  Откройте в браузере адрес `http://localhost:8000`.

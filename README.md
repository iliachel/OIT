# TinyML на микроконтроллерах

Сайт-визитка по реферату  
**«TinyML на микроконтроллерах: архитектуры, ограничения и сравнение фреймворков»**

## Автор
Челевич Илья Михайлович  
Белорусский государственный университет  
Специальность: Кибербезопасность

## Описание проекта
Многостраничный сайт, размещённый на GitHub Pages, посвящённый анализу
концепции TinyML и сравнению популярных фреймворков для микроконтроллеров.

## Используемые технологии
- HTML5
- CSS3
- JavaScript
- AOS (Animate On Scroll)
- GitHub Pages

## Запуск
Проект является статическим. Для просмотра достаточно открыть `index.html`
или перейти по ссылке GitHub Pages.

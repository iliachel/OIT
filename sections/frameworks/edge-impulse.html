Edge Impulse

Edge Impulse — это комплексная облачная платформа, предназначенная для упрощения всего цикла разработки моделей машинного обучения для встраиваемых систем. Она предоставляет инструменты для сбора, разметки и анализа данных с сенсоров, обучения моделей (включая AutoML) и их оптимизации для развёртывания на микроконтроллерах с низким энергопотреблением. Платформа ориентирована на быстрое прототипирование и позволяет разработчикам без глубоких знаний в области ML создавать готовые к использованию решения, сокращая время выхода на рынок.
